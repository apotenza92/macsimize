Macsimize for macOS

Macsimize turns the green window button into Maximize instead of Full Screen, keeps windows in your current space, overrides title-bar double-click, and adds Maximize All and Restore All in the menu bar.

Download
Or install via Homebrew:
brew tap apotenza92/tap && brew install --cask apotenza92/tap/macsimize
Beta Version: Beta builds install as Macsimize Beta.app and track the newest beta-channel release.

Requirements: macOS 14.0 or later, plus Accessibility and Input Monitoring permissions after first launch.
